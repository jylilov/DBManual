Обучающая система по базам данных представляет собой набор теоретических сведений по реляционным базам данных и языку запросов SQL.

Вся теория разделена на тематические уроки. Список уроков находится слева на панели инструментов. Для каждого урока существует некоторый набор задач для закрепления теоритического материала и получения практических навыков работы с базами данных. Задачи имеют разную сложность. Для каждого пользователя подбираются задачи соответствующие его уровню. Система определяет уровень пользователя по его результатам работы с системой. Для того, чтобы успешно пройти проверку знаний необходимо выполнить 5 задач.

Также Вы можете пробовать свои силы выполняя задания сразу по всем темам. Для этого в меню "Проверка знаний" выберите пункт "По всем темам". В этом случае необходимо выполнить 10 задач.

Каждая задача представляет собой описание таблиц и связей между ними, а также формулировку задания, которое необходимо выполнить написав SQL запрос. При проверке написсанного SQL запроса показывается результат его работы в виде таблицы. При правильном вводе, задание засчитывается сделанным и можно перейти к следующему заданию.
